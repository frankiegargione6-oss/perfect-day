# Perfect Day v0.17 Supabase Setup

1. In Supabase, open **SQL Editor**.
2. Run `supabase_update.sql`.
3. In Supabase, go to **Authentication → Providers → Email** and make sure Email is enabled.
4. For easiest testing, turn off email confirmation first. You can turn it back on later.
5. Push this project to GitHub. Render should redeploy automatically.

The Supabase URL and anon public key are already in `script.js` for this prototype.
