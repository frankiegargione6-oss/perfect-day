# Perfect Day v0.24

## New in v0.24

- Expanded the city/METAR pool from 94 to 142 locations.
- Rebuilt the game mode list:
  - Everyday
  - Summer
  - Winter
  - Beach
  - Golf
  - Tailgate
  - Severe Weather
  - Worst Day
- Added unique weighted scoring for each mode using the six current draft variables:
  - Temperature
  - Dew Point
  - Wind Speed
  - Sky Cover
  - Visibility
  - Pressure
- Worst Day now scores the most uncomfortable conditions instead of a normal ideal target.
- Leaderboard, Head-to-Head, Group Play, and Team Mode mode selectors now use the new modes.

No new Supabase SQL is required for this update.
